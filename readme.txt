== Description ==

Удалённая установка ваших плагинов.

Как это работает?:

Укажите полный URL к директории где храняться все ваши плагины. (Например: https://example.com/my_plugins/).
В эту директорию разместите 2 директории с плагинами и темами:

1. plugins
2. themes

Разместите ваши палгины и темы в .zip в соответствующие директории.

Этот плагин отобразит весь список тем и плагинов из директорий и вы сможете устанавливать нужные плагины и темы.
